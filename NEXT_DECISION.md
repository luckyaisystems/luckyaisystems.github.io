## Repository Philosophy (Version 2)

LuckyAISystems should function as a professional engineering knowledge system.

- improve judgment  
- reduce repeated mistakes  
- accelerate learning  
- preserve reusable knowledge  
- support long‑term career growth

---

## Knowledge Filter

Before creating or refining a page, apply the governing principle through four questions:

### 1. Is this reusable?
If not, keep it in the daily journal.

### 2. Does this change future decisions?
If not, archive as reference only.

### 3. Does this duplicate existing knowledge?
If yes, refine the existing page instead.

### 4. Is this the simplest structure that solves the problem?
Prefer refinement over expansion.

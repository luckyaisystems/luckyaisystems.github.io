## 5. Extraction Rule

Use this filter on every idea:

- Does it help me **communicate better**?  
- Does it help me **operate better at Equinix**?  
- Does it help me **save, plan, or buy property**?  
- Does it become a **reusable system** instead of a one‑off note?

If the answer is yes to at least two, it belongs in LuckyAISystems.
